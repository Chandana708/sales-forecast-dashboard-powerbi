# Sales Forecast & Dashboard (Power BI)

## Overview
This project presents an interactive Sales Dashboard along with a 15-day Sales Forecast built using Power BI.

It helps analyze past performance and predict future sales trends.

---

## Objectives
- Analyze sales, profit, and orders
- Track sales trends over time
- Forecast sales for the next 15 days
- Identify high and low performing regions

---

## Tools Used
- Power BI
- CSV Dataset

---

## Dataset
- File: "SalesDB.csv"
- Source: Kaggle Superstore Dataset

---

## Data Processing
- Imported dataset directly into Power BI
- Created visualizations and forecast using built-in features

---

## Dashboard Features
- KPI Cards (Sales, Profit, Orders)
- Sales by Category & Segment
- Sales Trends Over Time
- Regional Analysis

---

## Forecast Features
- 15-Day Sales Forecast
- Trend-based prediction using Power BI

---

## Conclusion
This project demonstrates the use of Power BI to transform raw data into insights and perform forecasting for better decision-making.
